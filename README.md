# Flowcharting for Grafana

## About
Flowcharting is a plugin for grafana, it allows to display complexe diagrams drawn with [draw.io](https://draw.io/) like :
  * Technical architecture schema.
  * Diagrams.
  * Organic plans.
  * Floorplans.
  * ULM plan.
