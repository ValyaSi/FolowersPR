# Flowcharting for Grafana

## About
Flowcharting is a plugin for grafana, it allows to display complex diagrams drawn with [draw.io](https://draw.io/) like :
  * Technical architectural schema.
  * Diagrams.
  * Organic plans.
  * Floorplans.
  * ULM plan.
